# 📊 Sales Analytics Dashboard

A comprehensive Sales Analytics Dashboard built using **Python, Pandas, Plotly, and Streamlit** to analyze sales performance, customer behavior, and regional trends.

## 🚀 Features

* Data Cleaning & Preprocessing
* Revenue Analysis
* Sales Trend Analysis
* Top Products Analysis
* Country-wise Revenue Analysis
* Customer Insights
* Interactive Filters
* Download Cleaned Dataset

## 📁 Dataset Columns

* InvoiceNo
* StockCode
* Description
* Quantity
* InvoiceDate
* UnitPrice
* CustomerID
* Country

## 📈 KPIs

* Total Revenue
* Total Orders
* Total Customers
* Total Quantity Sold
* Average Order Value

## 📊 Visualizations

* Revenue Trend Over Time
* Top 10 Products by Revenue
* Country-wise Sales Analysis
* Revenue Distribution
* Top Customers

## 🛠️ Technologies Used

* Python
* Pandas
* Plotly
* Streamlit

## ▶️ Run the Dashboard

```bash
streamlit run app.py
```
